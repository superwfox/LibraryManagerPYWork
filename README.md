# 大学图书馆学生借阅关系网络分析

## 项目简介

本项目旨在分析大学图书馆学生的借阅行为模式，重点关注学生之间的共借阅关系。
如果两个学生借阅过同一本书，则认为这两个学生之间存在一次“共借阅”关系，这种关系是相互的。
通过分析学生借阅网络，可以帮助图书馆了解学生借阅习惯，找出核心学生，并提供数据支持进行管理与决策。

项目使用 **Python 3** 开发，采用多文件模块化结构，便于扩展和维护。

## 功能说明

1. **构建借阅关系网络**

   * 读取 CSV 格式借阅记录数据；
   * 生成学生之间的共借阅关系网络。

2. **计算关联度**

   * 统计每个学生与其他学生的共借阅次数；
   * 去除重复关联，确保准确性。

3. **找出核心学生**

   * 关联度最高的学生被认定为网络核心学生；
   * 输出核心学生信息及其共借阅关系。

4. **生成分析报告**

   * 将分析结果写入 `result.txt` 文件；
   * 包含每个学生的关联学生列表及关联度统计。

```
LibraryAnalysis/
│
├─ FileManager.py       # 文件读取/写入与基础数据组织

├─ SystemIntro.py       # 主程序入口，调用各模块生成报告
├─ data.csv             # 示例借阅记录文件
├─ result.txt           # 分析结果输出文件
```

## 使用说明

1. **准备数据文件**

   * 在 `data.csv` 中按以下格式记录学生借阅信息：

     ```
     张三,001
     李四,001
     王五,002
     张三,003
     ```
   * 每行包含学生姓名和书号，中间用逗号分隔。

2. **运行程序**

   * 在 PyCharm 或命令行中运行 `SystemIntro.py`：

     ```bash
     python SystemIntro.py
     ```
   * 程序会自动读取 `data.csv` 并生成分析报告 `result.txt`。

3. **查看分析结果**

   * 打开 `result.txt`，可看到：

     * 每个学生的共借阅学生列表；
     * 每个学生的关联度；
     * 网络核心学生及其共借阅网络。

## 数据结构说明

| 模块          | 数据结构      | 用途说明                  |
| ----------- | --------- | --------------------- |
| FileManager | List      | 读取每行借阅记录，保持顺序，便于处理    |
| FileManager | Dict      | 学生/书籍映射关系，快速查找        |
| DataHandler | Set       | 去重共借阅关系，确保每个学生只统计一次关联 |
| SystemIntro | List/Dict | 存储最终分析结果并输出报告         |

## 项目特点

* 模块化设计，易于维护和扩展；
* 支持多学生、多书籍共借阅关系分析；
* 输出结果清晰、可读；
* 第一次完整使用 PyCharm 实现多文件协作项目，掌握了 Python 文件读写、字典/集合操作和数据分析流程。

---

## 注意事项

1. 数据文件必须为 UTF-8 编码，避免中文乱码；
2. 学生姓名和书号请保证正确且不含多余空格；
3. 数据量过大时，程序运行时间可能增加，可考虑优化算法或使用数据库处理。
